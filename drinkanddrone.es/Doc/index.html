Instalación de la WEB.

  1. Después de descomprimir el paquete de descarga, encontrarás una Carpeta llamada DrinkandDrone.es con todos los archivos.
  2. No podras visualizarla en tu navegador, ya que necesitaras un servidor web para como WAMP por ejemplo.
  3. No funcionará ya que contiene código PHP y requiere un entorno de servidor para enviar mensajes.
  4. Ahora ve a tu cPanel o abre tu Cliente FTP (como Filezilla) y sube el contenido del Tema a la raíz de tu servidor. Esto lo he conseguido contratando el dominio y el hosting en Hostalia.
  5. Una vez que los archivos hayan terminado de subir, ve a www.drinkdrone.es /// www.drinkanddrone.es

Libreria de PHP

Estos son los archivos de PHP que se cargan en las librerias PHPMailer-master Estan en la raiz del proyecto. 



Carpeta PHPMailer-master: Esta es una carpeta que contiene la librería PHPMailer. 
La carpeta PHPMailer-master es probablemente una versión descargada de la librería PHPMailer desde un repositorio como GitHub. 
Esta librería es ampliamente utilizada en PHP para enviar correos electrónicos de manera automatica. 
Los archivos contenidos en esta carpeta incluirán scripts PHP y, posiblemente, otros archivos necesarios para su funcionamiento, 
como plantillas de correo electrónico, archivos de configuración, entre otros.

En resumen, la librería de PHP contenida en la carpeta PHPMailer-master proporciona funcionalidades para enviar correos electrónicos desde un servidor web utilizando PHP, 
y los archivos en esta carpeta son los componentes necesarios para que esta librería funcione correctamente en un proyecto web.
